== Description ==

This plugin will let you the option to choose different themes for each browser.
Meanwhile,the plugin have only the options to choose another theme for the following browsers:IE6,IE7,IE8,FF2,FF3,Opera,Chrome,Safari and to all the other browsers which is not have their own option.

== Installation ==

1. Upload `browserthemer` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Which browser do the plugin support? =

IE6,IE7,IE8,FF2,FF3,Opera,Chrome,Safari

= How do i edit the plugin settings? =

Simply go to Dashboard>Settings>BrowserThemer
